# Divs and Sections Demo

This project demonstrates the use of **HTML `<section>` and `<div>` elements** with inline CSS styling. It is intended for beginners learning how to structure web pages and visually separate content.

## Features

- Two main `<section>` elements with distinct borders
- Multiple `<div>` elements inside each section
- Inline CSS for borders, widths, and heights
- Headings (`<h1>`) and paragraphs (`<p>`) inside divs
- Visual representation of how sections and divs work
